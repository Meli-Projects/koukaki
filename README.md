# Koukaki – Thème enfant FOCE
Projet OpenClassrooms – Projet 9  
Amélioration du site du studio d’animation Koukaki.

## Prérequis
- WordPress installé localement
- Node.js v16.x
- npm

## Installation
1. Cloner le dépôt dans le dossier :
   wp-content/themes/
2. Activer le thème **foce-child** depuis l’administration WordPress.
   
## Installation des dépendances et lancement de Sass
Dans le dossier du thème enfant :

```bash
npm install
npm run sass
